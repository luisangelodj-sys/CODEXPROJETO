# Roteiro de Video: Wyndham Corumba Lake Resort

Objetivo: aquecer publico frio/morno e gerar conversa no WhatsApp.  
Canal: Reels, Stories, Facebook Feed, WhatsApp e remarketing.  
Duracao recomendada: 15 segundos.  
Formato: vertical 9:16.

## Veredito estrategico antes do video

Este anuncio nao deve comecar vendendo "R$ 95.000" de forma seca para publico frio. O preco chama atencao, mas tambem pode gerar curiosidade ruim e confusao: a pessoa pode nao entender se e diaria, apartamento inteiro, cota, entrada ou fracao de multipropriedade.

A melhor abordagem para WhatsApp + aquecimento e:

1. Abrir com destino e alto padrao.
2. Explicar que e multipropriedade.
3. Mostrar que existe uma condicao de lancamento.
4. Chamar para entender pelo WhatsApp, sem promessa de investimento.

## Versao recomendada do video

### Cena 1 - 0s a 3s

Imagem: vista do lago/resort.  
Texto na tela:

```
Um novo jeito de viver ferias em Goias
```

Narração:

```
Imagine ter ferias planejadas todos os anos, em um resort as margens do Lago Corumba.
```

Funcao:
Abrir desejo sem comecar pelo preco.

### Cena 2 - 3s a 6s

Imagem: fachada do resort.  
Texto na tela:

```
Wyndham Corumba Lake Resort
Luziania - GO
```

Narração:

```
O Wyndham Corumba Lake Resort chega com estrutura de alto padrao em Luziania, proximo a Brasilia.
```

Funcao:
Dar localizacao e autoridade de marca.

### Cena 3 - 6s a 9s

Imagem: area interna/premium.  
Texto na tela:

```
Multipropriedade
use por temporada, sem comprar uma unidade inteira
```

Narração:

```
O modelo e multipropriedade: voce adquire uma fracao de uso, com regras e periodos definidos.
```

Funcao:
Reduzir confusao. Nao vender como diaria nem apartamento integral.

### Cena 4 - 9s a 12s

Imagem: arte com preco.  
Texto na tela:

```
Condicoes de lancamento
unidades a partir de R$ 95 mil*
```

Narração:

```
As condicoes de lancamento partem de 95 mil reais, conforme disponibilidade e regulamento.
```

Funcao:
Usar preco com mais seguranca.

### Cena 5 - 12s a 15s

Imagem: lago/resort ou fundo limpo com logo.  
Texto na tela:

```
Quer entender se faz sentido para voce?
Fale no WhatsApp
```

Narração:

```
Clique no WhatsApp e entenda como funciona antes de decidir.
```

Funcao:
CTA consultivo, ideal para aquecimento.

## Observacoes obrigatorias para rodar trafego

- Validar autorizacao de uso da marca Wyndham.
- Confirmar se as imagens representam o empreendimento real/render aprovado.
- Confirmar se o preco "a partir de R$ 95.000" se refere a fracao/cota/unidade de multipropriedade.
- Informar no destino do clique: disponibilidade, condicoes, taxas, regras de uso, calendario e contrato.
- Nao usar promessa de rentabilidade, valorizacao ou investimento garantido.
- Usar "*consulte condicoes, disponibilidade e regulamento" quando o preco aparecer.
